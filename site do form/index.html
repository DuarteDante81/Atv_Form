Questionário

  • 1-Na tag <input> com atributo type="text", para que serve o atributo "placeholder"?
    • O atributo placeholder é usado em input fields em formulários de HTML para exibir um texto temporário que descreve o tipo de dados que deve ser inserido pelo usuário no campo.
  • 2-Na tag <input> com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?
    • O atributo “checked” é usado na tag <input> com atributo type=”radio” ou type=”checkbox” para indicar que uma opção deve ser selecionada por padrão.
  • 3-Na tag <input> para que serve o atributo "value"?
    • O atributo “value” na tag <input> é usado para definir um valor padrão para o campo de entrada. Isso significa que quando a página for carregada, o campo já terá um valor pré-definido atribuído a ele.
  • 4-Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?
    • Cada botão de rádio deve ter o mesmo atributo "name", o que permite que eles sejam agrupados juntos em um conjunto. Quando o usuário seleciona um botão de rádio, ele é marcado como selecionado e os outros botões do mesmo conjunto são desmarcados automaticamente. Isso garante que apenas um botão de rádio seja selecionado de cada vez.
  • 5-Para que serve a tag <label>? Para que serve o atributo "for" nessa tag?
    • O elemento <label> define um rótulo para vários elementos de formulário. Ele consegue ser extremamente útil para os usuários do leitor de tela,porque o leitor irá ler os rótulos quando o usuário focar no elemento entrada. O atributo "for" faz a união do elemento <label> e o elemento <input>.
  • 6-Na tag <option> para que serve o atributo "selected"?
    • O atributo "select" serve para definir uma opção pré-selecionada.
  • 7-Nas tags de formulário para que serve o atributo "required"?
    • O atributo "required" nas tags de formulário é usado para indicar que um campo é obrigatório e deve ser preenchido antes que o formulário possa ser enviado.
  • 8-Para que serve a tag <textarea>?
    • A tag <Textarea> permite o usuário para escrever dentro de uma área programada.
  • 9-Na tag <form> para que serve o atributo "action"?
    • O action é um atributo que define a ação a ser executada quando o formulário é enviado.
  • 10-Na tag <form> para que serve o atributo "method"?
    • O method atributo especifica o método HTTP a ser usado ao enviar os dados do formulário. O método HTTP para enviar dados de formulário pode ser GET ou POST
  • 11-Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag <form>?
    • Deve-se usar o valor POST no atributo method
  • 12-Qual valor deve ser informado no atributo "method" na tag <form> em um formulário que será usado para fazer upload de arquivos?
    • Deve-se usar o valor GET no atributo method
  • 13-O que as tags <fieldset> e <legend> produzem?
    • A tag <fieldset>, permite criar um conjunto de campo. Já o <legend>, irá fazer uma legenda no sentido de rotular
